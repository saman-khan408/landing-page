# Portfolio Landing Page

A modern and responsive **portfolio landing page** built with **HTML**, **CSS**, and **Bootstrap 5**. This landing page is designed to showcase personal projects, skills, and professional information in a clean, mobile-friendly layout.

Live Demo: [https://saman-khan408.github.io/landing-page/](https://saman-khan408.github.io/landing-page/)

---

## Features

- Fully responsive and mobile-first design
- Hero section with introduction
- Project showcase section
- About Me section
- Contact form
- Smooth scrolling and interactive elements
- Easy to customize for your own portfolio

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
